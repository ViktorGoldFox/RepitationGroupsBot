# Repitation Groups Bot

Telegram-бот для репутации в группах.

## Возможности

- `+реп` - повысить репутацию пользователю в ответ на его сообщение
- `-реп` - понизить репутацию пользователю в ответ на его сообщение
- `/setrep <число>` - установить точную репутацию пользователю
- `топ реп` - показать топ участников группы
- автоматическая регистрация групп и пользователей в базе
- логирование действий и ошибок
- ограничение на `+реп` и `-реп`: не более 3 раз за 5 минут
- `/setrep` доступна только администраторам группы

## Команды

### `/start`
Показывает краткое описание бота и список доступных команд.

### `/help`
Показывает справку по командам и ограничениям.

### `+реп` / `-реп`
Используются только в ответ на сообщение пользователя.

### `/setrep <число>`
Используется только в ответ на сообщение пользователя и только администраторами группы.

### `топ реп`
Показывает топ-5 участников группы по репутации.

## Требования

- Python `3.12`
- SQLite
- Telegram-бот

## Запуск

1. Установи зависимости:

```bash
pip install -r requirements.txt
```

2. Проверь настройки бота в `Core/configs.py`.

3. Запусти приложение:

```bash
python main.py
```

## Настройка Telegram

Для работы бота в группе нужно:

- добавить бота в группу;
- отключить privacy mode у бота в BotFather, чтобы бот видел обычные сообщения `+реп`, `-реп` и `топ реп`.

## Хранение данных

При запуске создаются:

- `database.db` - база данных SQLite;
- `logs.log` - лог-файл приложения.

Оба файла игнорируются `git`.

## Конфигурация

Основные настройки лежат в `Core/configs.py`:

- `token` - токен Telegram-бота;
- `logs_path` - путь к логам;
- `db_path` - путь к SQLite базе;
- `echo` - вывод SQL-запросов в лог.

## Структура проекта

- `main.py` - точка входа
- `bot/` - роутеры, фильтры, middleware и бизнес-логика
- `database/` - модели и инициализация БД
- `Core/` - настройки

## Примечания

- Бот работает через reply-механику: репутация меняется только в ответ на сообщение пользователя.
- Если пользователь без `username`, в сообщениях выводится его `full name`.
